A spin on Connect 4 :)
=====================================================================================

Made by "257% Korean" in August 2020 for UniJam 2020 (https://unijam.org.au/),
a week-long game development hackathon at the University of Melbourne.

Jonathan Jauhari
Jun Hee Han
Sooyoung Jung
Tianchen Lou

---------------------------
Running the executable
---------------------------


---------------------------
Running from source
---------------------------

1. Download Python 3.7+
2. Run the following commands, in a virtual environment (conda/venv), or otherwise:

   pip install -r requirements.txt
   python connect4plus.py
